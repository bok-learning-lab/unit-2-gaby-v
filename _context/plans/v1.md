# Student Connection Platform — v1 Plan

## Overview

A two-tab web app for students to discover classmates and build connections. Tab one is an interactive world map where students pin their hometown with a personal profile. Tab two is a "We're Not Really Strangers"-style card game with tiered conversation prompts.

---

## Tab 1: Hometown Map

### Core Map Experience
- Interactive world map (Mapbox GL JS or Leaflet + OpenStreetMap)
- Each student places a pin on their hometown city
- Clicking a pin opens a profile card with:
  - Name, photo (optional), hometown name
  - Class year, concentration/major, dorm/house
  - 3–5 fun facts (free-form, user-written)
  - Optional: favorite spots on campus, a "ask me about" field

### Student Profile / Pin Creation
- Simple onboarding form to submit your pin:
  - Hometown (geocoded via search, not manual lat/lng)
  - Personal details: name, class year, concentration, dorm
  - Fun facts (e.g. "I've visited 14 countries", "I'm a competitive chess player")
  - Optional photo upload
- Edit your own pin after submission (auth required)

### Filters (sidebar or top bar)
- Class year (e.g. 2025, 2026, 2027, 2028)
- Concentration / major
- Dorm or residential house
- Filters update visible pins in real time; filtered-out pins dim or hide
- "Clear filters" button

### Map UX Details
- Cluster overlapping pins at low zoom levels (show count badge)
- Expand clusters on zoom
- Hover on pin: show name + hometown tooltip
- Click pin: open full profile card (modal or side drawer)
- Search bar to find a specific student by name

## Tab 2: Connection Prompts

Inspired by *We're Not Really Strangers*. Questions are organized into three levels — meant to be used between two students who just connected on the map or met in person.

### Level Structure

**Level 1 — Icebreakers (Surface)**
Light, easy, low-stakes. Good for a first conversation.
Examples:
- "What's something you're really proud of that most people here don't know about?"
- "If you could instantly master one skill, what would it be?"
- "What's the most interesting place your hometown is close to?"

**Level 2 — Getting Real (Middle)**
Reflective, a bit more personal. Good once you've broken the ice.
Examples:
- "What's something you assumed about college that turned out to be completely wrong?"
- "What's a value you hold that surprises people when they find out?"
- "What are you most afraid of getting wrong in the next four years?"

**Level 3 — Deep Cut (Vulnerable)**
Meaningful, honest, not for strangers. Good when you want to actually know someone.
Examples:
- "What's a version of yourself you've had to let go of to be here?"
- "What do you wish people asked you more about?"
- "What's something you've never told anyone at this school?"

### Prompt UI
- Card-flip interaction (front: level badge + tap prompt, back: the question)
- Shuffle / draw a new card within the same level
- "Play together" mode: two people on separate phones, same session — each gets the same prompt simultaneously (websocket or shareable link with query param)
- Optionally: a daily featured prompt surfaced on the homepage

## Tech Stack (Recommended)

| Layer | Choice | Reason |
|---|---|---|
| Frontend | Next.js (App Router) | SSR, easy routing, good DX |
| Map | Mapbox GL JS | Clustering, custom markers, smooth UX |
| Styling | Tailwind CSS | Fast iteration |
| Auth | Clerk or NextAuth | Simple student login (email or Google) |
| Database | Supabase (Postgres) | Realtime, easy geo queries, free tier |
| Geocoding | Mapbox Geocoding API | Pairs with map, free up to 100k req/mo |
| Hosting | Vercel | Zero-config Next.js deploys |

## Data Model (Simplified)

```
Student
  id, user_id, name, photo_url
  hometown_name, lat, lng
  class_year, concentration, dorm
  fun_facts: string[]
  ask_me_about: string
  created_at, updated_at

Prompt
  id, level (1|2|3), text
  created_at
```

## Pages / Routes

```
/               → landing / map tab (default)
/map            → full map view with filters
/profile/[id]   → individual student profile (shareable link)
/prompts        → prompt card tab
/onboarding     → add your pin form
/edit           → edit your own pin
```

---

## MVP Scope (What to Build First)

1. Map with hardcoded/seeded student pins
2. Pin profile card (click to open)
3. Basic filters (class year, concentration, dorm)
4. Prompt cards — all three levels, single-player mode
5. Auth + onboarding form to add your own pin

**Defer to v2:**
- "Play together" realtime mode
- Photo uploads
- Daily featured prompt
- Mobile-native feel / PWA
